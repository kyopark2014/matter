# matter

매터(Matter)는 [CSA (Connectivity Standards Alliance)](https://csa-iot.org/all-solutions/matter/)에서 만든 표준으로, 홈디바이스와 같은 IoT 디바이스들이 쉽게 연결될 수 있도록 하는 오픈 소스 기반의 응용계층(application layer) 연결 표준입니다. IP 기반으로 동작하며 WiFi 네트워크를 지원합니다.
